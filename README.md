## Description
This project is an API built using **Spring Boot**.  
It demonstrates basic CRUD operations and proper HTTP status handling.

The API allows clients to:
- Create products
- Read products
- Update products
- Delete products

---

## HTTP Methods

1. **POST** – CREATE  
2. **GET** – READ  
3. **PUT** – UPDATE  
4. **DELETE** – DELETE  

---

## HTTP Status Codes

- **200 OK** – Successful request  
- **300** – Redirection responses  
- **400 Bad Request** – Client-side errors (e.g. invalid input)  
- **500 Internal Server Error** – Server-side errors
